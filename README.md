# Heart-Attack-Cases-in-India-Dashboard
An interactive healthcare analytics dashboard designed to analyze heart attack cases using patient demographics, lifestyle factors, and clinical indicators.

Heart attacks remain one of the leading causes of death worldwide.
Hospitals and health analysts often struggle to identify risk patterns across age, gender, region, lifestyle, and medical history using raw patient data.

The challenge is to transform complex clinical datasets into clear, actionable insights that support prevention, early diagnosis, and policy decisions.

ğŸ’¡ Solution Overview

This project delivers a visual analytics dashboard that summarizes heart attack cases using key health metrics and categorical risk factors.

The dashboard enables users to:

Compare affected vs non-affected patients

Analyze risk across age, blood pressure, cholesterol, and blood sugar

Understand lifestyle impacts such as exercise and smoking

Identify regional and gender-based patterns
